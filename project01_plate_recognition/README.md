## CV项目1：车牌识别项目任务

1. 根据参考代码，实现对输入图像进行车牌识别， 其中输入图像可自我采集，至少包括以下几种情况，并对识别不成功的情况分析失败原因。
    - 能成功识别的车牌图
    - 一幅图有多个车牌
    - 新能源（绿色底）车牌
    - 车牌倾斜、被遮挡、光线暗等3种情况

2. 问题分析
   - 问题1： 分析原车牌识别方法，哪些模块可以用机器学习的方法替换？
   - 问题2： 通过编程实现，比较实验结果，阐述机器学习模型相对于传统模型的优势？

3. 编程实践

    在车牌定位模块，编程实现用一种机器学习方法替换原传统方法，以实现车牌的定位。
